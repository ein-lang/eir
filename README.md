## Install

In your `Cargo.toml`,

```
eir = { git = "https://github.com/raviqqe/eir", branch = "master" }
eir-fmm = { git = "https://github.com/raviqqe/eir", branch = "master" }
```

## Features

- Currying
- Nominal typing
- Open variant type
- Foreign function interface
